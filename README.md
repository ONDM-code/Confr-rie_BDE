# Confrérie BDE

Bienvenue sur le projet **Confrérie BDE** ! Ce projet consiste à créer un site vitrine responsive qui présente l’univers d’une confrérie (réelle ou fictive) en mobilisant les notions apprises en HTML et CSS.

> ⚠️ **Ce projet a été réalisé dans le cadre d’un travail de classe.**

## 🚀 Objectifs du projet

- Créer un site vitrine moderne, responsive et attractif
- Présenter l'univers et les activités d'une confrérie
- Mettre en pratique les compétences en HTML et CSS
- Intégrer les composants suivants :
  - Un tableau (ex : planning, membres, événements…)
  - Un formulaire de contact
  - Menus de navigation
  - Galerie d’images, textes, etc.

## 🛠️ Structure du projet

- `index.html` : Page d'accueil
- `styles/` : Dossier contenant les fichiers CSS
- `images/` : Dossier contenant les images du site
- `contact.html` : Page du formulaire de contact
- `evenements.html` : Tableau des événements/précédentes réunions

## 📦 Installation

1. **Cloner le dépôt**
    ```bash
    git clone https://github.com/ONDM-code/Confr-rie_BDE.git
    cd Confr-rie_BDE
    ```
2. **Ouvrir le fichier `index.html` dans votre navigateur préféré.**

Aucune installation supplémentaire n’est requise, le site utilise uniquement HTML et CSS.

## ✨ Fonctionnalités

- Site responsive compatible mobile, tablette et desktop
- Navigation simple et intuitive
- Présentation dynamique de la confrérie (histoire, membres, événements…)
- Formulaire de contact opérationnel (HTML uniquement)
